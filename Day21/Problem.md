# Day 21: Longest Palindromic Substring

## Problem Statement
- Given a string s, return the longest palindromic substring in s.

## Example

1. **Example 1:**
- Input: s = "babad"
- Output: "bab"
- Explanation: "aba" is also a valid answer.

2. **Example 2:**
- Input: s = "cbbd"
- Output: "bb"
